# Contributing

1. Repo fork karo
2. Apna branch banao: `git checkout -b feature/naya-feature`
3. Changes karo aur commit karo: `git commit -m 'Naya feature add kiya'`
4. Push karo: `git push origin feature/naya-feature`
5. Pull Request bhejo
